# Notification App Backend

This folder serves as a placeholder for the backend service. Since this task focuses on the Frontend Track, the backend codebase is omitted.

If a backend were implemented here, it would use:
- Node.js & Express / NestJS (for API routes)
- PostgreSQL / Prisma (for database storage)
- Redis (for caching and rate-limiting)
- Kafka / RabbitMQ (for event queues)
